Using SEEK

Copy and paste any Youtube URL

Use the transport controls to navigate, with keyboard controls:

[space]play/pause
[right]jump ahead five seconds
[shift + right arrow]jump ahead thirty seconds
[left arrow]jump back five seconds
[shift + left arrow]jump back thirty seconds
